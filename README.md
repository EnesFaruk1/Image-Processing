# Image-Processing

This is an image processing program for many filters and edge detections techniques.
There are the techniques:

- Sobel Detection
- Prewitt Detection
- Laplacian Transform
- Sharpen Filter
- Mean Filter
- Gaussian Filter
- 4 Bit Edge Detection
- 8 Bit Edge Detection
- Brightness Filter
- Contrast Filter
- RGB Filter
- Down Scale
- Up Scale
- Up Scale (Bilinear)
